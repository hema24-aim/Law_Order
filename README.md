Law & Order Support System for Rural Areas (LawOrderApp)
The LawOrderApp is a web-based application designed to improve law and order management in rural areas. It allows citizens to report complaints easily and enables authorities to monitor and resolve issues efficiently.
User Features:
Register and Login
Raise complaints
Generate unique Complaint ID
Track complaint status
Simple and user-friendly interface
Admin Features:
View all complaints
Monitor complaint status
Dashboard with real-time data
Tech Stack:
Frontend: React.js, CSS
Backend: Node.js, Express.js
Database: MongoDB
Future Enhancements: 
OTP verification
SMS alerts using Twilio
Geo-location tagging
Complaint filtering by village
Dashboard with charts
Admin authentication
